# Gestion des Salles

## Description
**Gestion des Salles** est une application conçue pour faciliter l’organisation et la gestion des salles dans un établissement comme les universités.  
Elle permet de suivre la disponibilité des salles, de gérer les réservations, les utilisateurs et les plannings, afin d’optimiser l’utilisation des espaces.

## Fonctionnalités
- Ajouter, modifier et supprimer des salles.
- Gérer les utilisateurs (étudiants, enseignants, personnel).
- Réserver des salles avec contrôle de disponibilité.
- Visualiser les plannings par salle et par utilisateur.
- Notifications simples pour les conflits de réservation.
